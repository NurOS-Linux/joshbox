# JoshBox

## О проекте

JoshBox — это легковесная коллекция основных Unix-утилит, разработанная с упором на минимализм и эффективность. Подобно BusyBox, проект нацелен на предоставление необходимых системных инструментов в компактном, оптимизированном пакете.

## Статус

⚠️ **Ранняя стадия разработки** - Проект находится в концептуальной фазе/планирования. Реализация в процессе.

## Планируемые утилиты

**Манипуляция данными и файлами**
- `ls` — Просмотр содержимого каталогов
- `cp` — Копирование файлов и каталогов
- `mv` — Перемещение/переименование файлов
- `rm` — Удаление файлов и каталогов
- `mkdir` — Создание каталогов
- `cat` — Объединение и вывод файлов
- `grep` — Поиск текстовых шаблонов
- `sed` — Потоковый редактор
- `chmod` — Изменение прав доступа
- `chown` — Изменение владельца файла

**Сетевые инструменты**
- `ip` — Настройка сетевых интерфейсов
- `dhcpcd` — DHCP-клиент
- `ss` — Статистика сокетов

**Мониторинг системы**
- `ps` — Статус процессов
- `kill` — Завершение процессов
- `top` — Монитор системных ресурсов
- `dmesg` — Буфер сообщений ядра

## Цели

1. **Чистота** — Никакого лишнего кода, только необходимый функционал
2. **Производительность** — Минимальные накладные расходы и максимальная эффективность
3. **Безопасность** — Современные, безопасные реализации без уязвимостей legacy-кода
