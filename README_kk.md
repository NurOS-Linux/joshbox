# JoshBox

**Busybox стиліндегі минималды утилиталар жинағы**

[English](README.md) • [Русский](README_ru.md)

---

## Жоба туралы

JoshBox — бұл минимализм мен тиімділікке баса назар аудара отырып жасалған жеңіл Unix утилиталарының жинағы. BusyBox сияқты, бұл жоба қажетті жүйелік құралдарды ықшам, оңтайландырылған пакетте ұсынуға бағытталған.

## Күйі

⚠️ **Әзірлеудің бастапқы кезеңі** - Жоба қазір тұжырымдамалық/жоспарлау фазасында. Іске асыру үдерісте.

## Жоспарланған утилиталар

**Деректер мен файлдарды басқару**
- `ls` — Каталог мазмұнын көрсету
- `cp` — Файлдар мен каталогтарды көшіру
- `mv` — Файлдарды жылжыту/қайта атау
- `rm` — Файлдар мен каталогтарды жою
- `mkdir` — Каталогтар құру
- `cat` — Файлдарды біріктіру және шығару
- `grep` — Мәтіндік үлгілерді іздеу
- `sed` — Ағындық редактор
- `chmod` — Қолжетімділік құқықтарын өзгерту
- `chown` — Файл иесін өзгерту

**Желілік құралдар**
- `ip` — Желілік интерфейстерді конфигурациялау
- `dhcpcd` — DHCP клиент демоны
- `ss` — Сокет статистикасы

**Жүйені мониторингтеу**
- `ps` — Процестер күйі
- `kill` — Процестерді тоқтату
- `top` — Жүйелік ресурстар мониторы
- `dmesg` — Ядро хабарламалары буфері

## Мақсаттар

1. **Тазалық** — Артық код жоқ, тек қажетті функционал
2. **Өнімділік** — Минималды шығындар және максималды тиімділік
3. **Қауіпсіздік** — Заманауи, қауіпсіз іске асырулар, ескі осалдықтар жоқ
